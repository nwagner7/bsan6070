CA 03 - Decision Tree AlgorithmAssignment OverviewIn this assignment, decision trees were trained, tested, and visualized to identify the best fitting model. The data was provided in the Brightspace assignment folder for CA03.Configuration/Installation InstructionsThis project was run on Google Colab. Several packages must be imported, which can be found inside the notebook. Files IncludedThis assignment contains an ipynb file, a README file, and a pdf document answering the written portion of the assignment.Known BugsIn section 6, the code to identify the AUC value does not execute correctly.
