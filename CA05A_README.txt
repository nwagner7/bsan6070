CA05A Ð Logistic RegressionAssignment OverviewIn this assignment, a data analysis is performed to identify the most influential features in determining cardiovascular disease (CVD). The data consists of real heart patients collected from a 15 year study.Configuration/Installation InstructionsThe code was written in a GoogleColab notebook and is in an ipynb file. The file can be viewed and opened in GoogleColab, Jupyter Notebook, or any other known platform for viewing code. The dataset used was provided in the Brightspace folder and can be found at this link: https://brightspace.lmu.edu/d2l/le/content/132718/viewContent/1497201/ViewFiles IncludedThis assignment includes three files: the code in ipynb format, a pdf document discussing findings from the analysis, and a README file.Known BugsThere are no known bugs in the code.
